# vector容器
1. 数据结构：连续内存空间，顺序存储，单向开口；
2. 迭代器：随即迭代器，本质普通指针
3. 空间配置器：
    1. 每次分配空间时，会分配比所需的大一点。
    2. 每次重分配，都会释放原来空间，重新申请拷贝。
    注意. 重分配空间后，原来的迭代器失效。
4. 代价：增长带来的重开辟空间

api：
1. 添加
2. 删除
3. 插入
4. 空间配置
