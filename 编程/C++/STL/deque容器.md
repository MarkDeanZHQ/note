# deque容器
1. 数据结构：连续线性空间，逻辑上连续，双向开口；
    1. 采取一块map(此处不是指map容器)作为中控器。
    2. 中控器的节点存有 缓存区的地址。
    3. 单个缓存区中的空间是连续性内存空间。
    4. 缓存区是 deque容器的主体。
2. 迭代器：随机迭代器，复杂度与vector非一个量级。
    1. 高效率建议：排序时，建议在vector上排好序，再拷贝给deque容器。
3. 代价：复杂的迭代器架构。
