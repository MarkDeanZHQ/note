# 函数指针

* 函数指针是一个 指向函数入口地址的 指针，
* 可以通过地址来调用函数
* 函数名  函数入口地址

可以在Qt中实现， Qt中地址不变化

函数指针的声明 ：
    函数三要素 
    1. 返回值类型
    2. 函数名
    3. 传入形参
    

## 函数指针的写法
```c
void(*pFunc)(int)
```
返回值类型 + （指针变量名）（形参列表）

定义方式： example:   
```c
void(\*func)(int ,char )= (void(\*)(int  ,char ))p;

如果知道地址，可直接传地址
void(\*func)(int ,char )=(void(\*)(int  , char ))2114344

调用 
func(12,'a');
```
