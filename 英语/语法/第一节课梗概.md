# 第一节课梗概

句子成分：
* 主语
* 谓语
* 宾语
* 表语
* 补语

五大基本类型
* 主谓
* 主谓宾
* 主谓宾宾
* 主谓宾补
* 主系表

名词从句
核心：把句子当成，名词
    根据放在句子中，作为句子成分的不同，分为不同的从句
- 主语从句
- 宾语从句
    - 特殊化：使用频率最高
    - 可以省略 that ，if 、whether 任用    (其他从句必须加 that 只能使用 whether)
- 表语从句
- 同位语从句
    - 用于抽象名词
    - 遇到 抽象名词 ，应条件反射想到 跟了同位语从句
        - 常用抽象名词：belief，fact，idea，problem，information，conclusion，decision，news，thought，explanation
    - 英语中忌讳头重脚轻
        - 同位语从句充当主语过长时，同位语从句和它所修饰名词常常隔开。
    

- 疑问句做 名词从句名词
    - 一般疑问句做名词
        - 还原陈述句语序，if、whether开头
    - 特殊疑问句作名词
        - wh .. 、how 开头，封装在 一般疑问句上
        - 还原陈述语序，wh.. how 保留
